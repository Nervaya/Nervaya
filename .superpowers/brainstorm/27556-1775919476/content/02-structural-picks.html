Which structural improvements matter most?

Nervaya keeps its purple theme. Which design system structures should we adopt to make it more professional? Select all that apply.

🔒 Locked: Purple accent (#7c3aed), gradients, dark navy backgrounds — these stay as-is.
1

Typography System

Current: Outfit + Inter fonts, basic size scale (0.75rem–3rem), no letter-spacing rules, inconsistent line-heights

Upgrade: Proper hierarchy table with weight rules, negative letter-spacing at display sizes, consistent line-height scale (tight for headings, relaxed for body), micro-tracking on labels. Inspired by Stripe/Notion's precision typography.

2

Shadow & Depth System

Current: Basic 2-layer shadows, inconsistent across components

Upgrade: Multi-layer shadow stacks (4-5 layers, max 0.05 opacity each) for natural depth. Purple-tinted shadows for brand consistency. Clear elevation levels: Flat → Whisper → Card → Elevated → Modal. Inspired by Stripe's blue-tinted shadows + Notion's layered approach.

3

Component Styling Rules

Current: Buttons, cards, inputs exist but styling varies per page

Upgrade: Codified button variants (primary, secondary, ghost, pill badge) with exact specs. Card system with consistent border, radius, padding. Input focus states with purple ring shadows. Hover/active/disabled states for everything. Inspired by Claude's ring-shadow buttons + Stripe's component precision.

4

Spacing & Layout Grid

Current: 8px base with 6 steps (4px–64px), basic breakpoints

Upgrade: Extended spacing scale (4px–120px) with section-level breathing room. Defined container widths, editorial pacing between sections, consistent card grid gaps. Whitespace philosophy — sections breathe like chapters. Inspired by Claude's editorial pacing + Notion's generous rhythm.

5

Border & Radius System

Current: Mixed radius values (8px–24px), basic borders (#e2e8f0)

Upgrade: Defined radius scale (4px sharp → 9999px pill) with clear usage rules. Whisper-weight borders (rgba-based, barely visible) for sophistication. Purple-tinted borders for interactive elements. Inspired by Notion's whisper borders + Claude's warm ring system.

✓

All of the Above

Full professional design system overhaul — adopt best practices across typography, shadows, components, spacing, and borders. The complete package.
